DisSlash Discord Bot

DisSlash is a next-gen Discord Bot that integrates slash commands with your server using Discord's Beta API.

Built using Discord's Python wrapper, as well as the slash command wrapper, I have built a Discord Bot that intergrates slash commands in server. DisSlash has a MongoDB database, and hosted on Heroku, 24/7. Some of the commands include Notepad, Giveaway, QR Code, Moderation, as well as Game Stats.

  • Client: Personal Project
  • Category: Bots

SimpleList

Python Package

SimpleList is a Python Package that allows users to easily intergrate txt files into your python project. Along with that, it comes with 3 inbuilt lists with in inclue yes and no key words, as well as tickers.

QuickQR

Simple interactive website that looks great on both mobile and computer.

QuickQR is a free website that allows anyone to easily generate QR Codes, as well as track Uses, Location & More. QuickQR is built on a Flask Framework and uses a MySQL DB.
